LLM 增强示例

pv_short

pv_short(uint8_t string_id, pv_protect_cfg_t *cfg) → int

【功能描述】
检测指定 PV 组串是否发生短路。若电流超过阈值，
触发保护动作：断开对应组串继电器并记录故障。

【调用前置条件】
1. adc_init() 已执行且 adc_calibrated() 返回 true
2. relay_driver_init() 已完成，继电器处于可控状态
3. 不得在 MPPT 扫描周期内调用，避免瞬态电流误判

// 调用者: power_ctrl_loop(), startup_self_check()
核心价值
函数签名告诉 AI「能调什么」，
前置条件告诉它「调之前必须先做什么」。

逆变器功率软件中，前置条件错误是最危险的一类 bug —— ADC 未校准就做短路检测， 阈值判断全部失效，轻则误触发停机，重则短路保护失灵。把隐性知识变成显式上下文，是 LLM 增强的核心意义。

传统方式：粘贴文件 → 覆盖不完整 + 精度不够
